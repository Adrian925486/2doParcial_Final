# 🐾 Mundo Mascotas - Sitio Web

Proyecto web desarrollado para la materia de Programación Web, demostrando el uso de HTML, CSS y JavaScript con múltiples funcionalidades interactivas.

## 📋 Características

### Páginas
- **Inicio**: Catálogo de mascotas con filtros por categoría
- **Nosotros**: Información del equipo y formulario de contacto
- **Servicios**: Servicios ofrecidos y carrito de compras

### Funcionalidades JavaScript
1. **Carrito de Compras**
   - Agregar productos/servicios
   - Eliminar items
   - Vaciar carrito
   - Cálculo automático de total

2. **Menú Responsive**
   - Botón hamburguesa para móviles
   - Navegación intuitiva

3. **Validación de Formularios**
   - Validación de campos requeridos
   - Validación de email con regex
   - Mensajes de error dinámicos

4. **Filtrado de Productos**
   - Filtrar mascotas por categoría

5. **Interactividad**
   - Resaltar tarjetas con toggle de clases CSS

## 🛠️ Tecnologías Utilizadas

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (Vanilla)
- DOM Manipulation
- Local Storage (para persistencia del carrito)
